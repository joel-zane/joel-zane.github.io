## 项目简介

一期一会是我的个人博客，用来记录学习、项目和生活。采用 Hugo 静态网站生成器 + Archie 主题，部署在自有的 Oracle Cloud ARM64 服务器上，同时在 GitHub Pages 做备份。

## 技术栈

- **框架**: Hugo（静态网站生成器）
- **主题**: Archie（极简风格）+ 自定义页面模板
- **样式**: 自定义 CSS（暗色主题 + 绿色强调色）
- **部署**: Nginx（主站）+ GitHub Pages（备份）
- **统计**: 不蒜子访问计数
- **CI/CD**: GitHub Actions 自动构建

## 功能

- 博客文章（Markdown 写作）
- 项目展示（分类卡片 + 详情页）
- 画廊（图片瀑布流）
- 时间线（个人成长轨迹）
- 工具箱（常用工具聚合）
- 关于页面（个人信息 + 技术栈）
- RSS 订阅
- 访问统计
