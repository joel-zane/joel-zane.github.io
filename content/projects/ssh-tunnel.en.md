## About

A Windows desktop application for managing SSH tunnels with a graphical dashboard, real-time monitoring, and automatic reconnection.

## Features

- **GUI Dashboard**: Visual tunnel management interface
- **Real-time Monitoring**: Live status and traffic stats
- **Auto-reconnect**: Automatic recovery from disconnections
- **System Tray**: Minimize to tray for background operation
- **Multiple Tunnels**: Manage several tunnels simultaneously

## Tech Stack

- **Language**: Python
- **GUI**: tkinter / PyQt
- **SSH**: Paramiko library
