## About

Hermes is a self-hosted AI assistant running on Oracle Cloud, connected via WeChat for daily interactions and automation.

## Features

- **WeChat Integration**: Chat via iLink Bot API
- **Daily Briefings**: Automated weather, news, and reflections
- **Web Automation**: Browsing, research, and data collection
- **Code Assistance**: Writing, debugging, and deployment
- **Task Scheduling**: Cron jobs and reminders
- **Multi-modal**: Vision, TTS, image generation

## Tech Stack

- **Runtime**: Node.js on Oracle Cloud ARM64
- **LLM**: MiMo (Xiaomi) / DeepSeek
- **Messaging**: WeChat via iLink Bot
- **Tools**: File management, web search, code execution, MCP servers
