## About

"Yi Qi Yi Hui" (一期一会) is a personal blog built with Hugo and the Archie theme. The name comes from the Japanese concept of "once in a lifetime" encounters.

## Features

- **Dark/Light mode** toggle
- **Bilingual** support (Chinese/English)
- **Gallery** with lightbox and zoom
- **Music player** with playlist switching
- **Waline** comment system
- **Umami** analytics

## Tech Stack

- **Framework**: Hugo with Archie theme
- **Comments**: Waline (self-hosted)
- **Analytics**: Umami (self-hosted)
- **Hosting**: Oracle Cloud ARM64 + GitHub Pages
