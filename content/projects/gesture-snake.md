## 项目简介

基于 NUCLEO-G474RE 开发板的体感贪吃蛇游戏。通过 MPU6500 加速度计检测倾斜方向控制蛇的移动，用力摇动重置游戏，画面显示在 1.8 寸 ST7735 TFT 彩色屏幕上。

## 硬件组成

| 组件 | 型号 | 接口 |
|------|------|------|
| 主控 | STM32G474RE (NUCLEO) | - |
| 屏幕 | 1.8寸 ST7735 TFT (128×160) | 软件 SPI |
| 传感器 | MPU6500 (加速度计+陀螺仪) | 软件 I2C |

## 核心功能

- **倾斜控制**: 倾斜板子改变蛇的方向，阈值可调 (TILT_ON=2600)
- **摇一摇重置**: 快速摇动陀螺仪检测，连续4次确认防误触
- **局部刷新**: 蛇移动时只刷新头/尾/食物，避免整屏闪烁
- **3条命系统**: 撞墙/撞身体扣命，0命 Game Over
- **自动重连**: MPU6500 通信失败时每 500ms 自动重试

## 游戏参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 棋盘 | 12×14 格 | 每格 10×10 像素 |
| 移动间隔 | 180ms | 可调，越小越快 |
| 倾斜阈值 | 2600 | 越大越不灵敏 |
| 初始长度 | 4 节 | 吃食物+1 |

## 文件结构

| 文件 | 说明 |
|------|------|
| `BSP/snake_game.c` | 游戏逻辑、碰撞检测、分数生命 |
| `BSP/st7735.c` | ST7735 TFT 驱动 (软件 SPI) |
| `BSP/mpu6500.c` | MPU6500 驱动 (软件 I2C) |
| `Core/Src/main.c` | 主循环：读传感器→更新游戏 |
